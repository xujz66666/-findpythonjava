# -findpythonjava
If the environment is installed, return this language (Python, Java); if not, return not found (Python, Java)
I put find_python.exe in FindPythonJava\dist. find_java.java is placed in FindPythonJava\FindPythonJava\src\find_java.java, and the .jar file is in FindPythonJava\FindPythonJava\out\artifacts\FindPythonJava
_jar\find_java.jar. 
Declaration: This program is not superfluous; it does not rely on the environment to check whether the user has installed it. If not, it returns 'not find python not find java', and if it is installed, it returns 'python, java'.
